⚙️ Installation

Clone the repository:

git clone https://github.com/Mrigank-loop/AI-Resume-Analyzer.git
cd AI-Resume-Analyzer

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
🎯 How It Works
Upload a resume in PDF format.
Paste a job description.
The system extracts text from both documents.
Text similarity is calculated using NLP techniques.
ATS score, missing keywords, and suggestions are displayed.
