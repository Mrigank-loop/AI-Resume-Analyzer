# AI Resume Analyzer

An ATS-powered Resume Analyzer built using Python and Streamlit.

## Features

- Resume PDF Upload
- ATS Match Score
- Missing Keyword Detection
- Skill Extraction
- Resume Improvement Suggestions

## Tech Stack

- Python
- Streamlit
- PDFPlumber
- Scikit-Learn
